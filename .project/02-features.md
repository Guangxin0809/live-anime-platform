# 功能模块规划

## Phase 1 — 基础架构与认证（MVP）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 用户注册 | 邮箱注册 + 手机号注册 | P0 |
| 用户登录 | 密码登录，JWT Token 管理 | P0 |
| 个人信息 | 头像、昵称、基本资料编辑 | P1 |
| Token 刷新 | 自动刷新 access token | P0 |
| 基础 UI 框架 | 布局、导航、全局状态 | P0 |

## Phase 2 — 动漫内容模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 动漫列表 | 分页浏览，支持封面展示 | P0 |
| 动漫详情 | 简介、剧集列表、演员表 | P0 |
| 搜索 | 关键词搜索动漫标题、标签 | P0 |
| 分类筛选 | 按类型、年份、地区筛选 | P1 |
| 标签系统 | 多标签关联与筛选 | P1 |
| 剧集播放页 | 选集、播放器嵌入 | P1 |

## Phase 3 — 支付与订单

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 购物车 | 选择内容、确认购买清单 | P0 |
| 下单流程 | 创建订单、选择支付方式 | P0 |
| 支付宝支付 | 扫码支付 / App 跳转支付 | P0 |
| 微信支付 | JSAPI / Native 支付 | P0 |
| 支付回调 | 异步通知处理、订单状态更新 | P0 |
| 订单列表 | 按状态（待支付/已支付/已取消）筛选 | P1 |
| 订单详情 | 查看订单商品、支付信息 | P1 |
| 退款处理 | 申请退款与状态跟踪 | P2 |

## Phase 4 — 后台管理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 管理员登录 | 独立后台认证 | P1 |
| 动漫管理 | 增删改查动漫及剧集 | P1 |
| 订单管理 | 查看/处理用户订单 | P1 |
| 用户管理 | 查看用户列表、封禁/解封 | P2 |
